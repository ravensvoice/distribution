Duke Nukem 3D:

DUKE3D.GRP (For proper support for World Tour's extra features requires the whole game installation)

********************

The following Duke games must be in separate folders with their own copy of DUKE3D.GRP. Not all versions of DUKE3D.GRP are compatible so if it crashes on start or just loads the standard Duke Nukem 3D game try it with a different DUKE3D.GRP. I experienced the best compatibility with the DUKE3D.GRP that had a CRC-32 File Checksum of fd3dcff1. You can check the checksum online at: https://emn178.github.io/online-tools/crc32_checksum.html

Duke It Out in D.C.:

DUKEDC.GRP (Best compatibility with CRC-32 File Checksum of a8cf80da)

Duke: Nuclear Winter:

NWINTER.GRP (Best compatibility with CRC-32 File Checksum of f1cae8e4)

Duke Caribbean: Life's a Beach:

VACATION.GRP (Best compatibility with CRC-32 File Checksum of 18f01c5b)
